FAQ

Le site stocke-t-il mes données ? +
Non. Toutes les informations restent sur votre appareil. Aucune donnée n’est envoyée à un serveur.
Les documents sont-ils valables légalement ? +
Ce sont des modèles standards. Ils doivent être adaptés selon votre pays, vos conventions collectives et vos règles internes.
Puis-je utiliser ce site pour mon entreprise ? +
Oui, totalement. Vous pouvez l’utiliser pour générer des documents RH de base pour votre organisation.
Le service est-il gratuit ? +
Oui, entièrement gratuit. Aucune inscription, aucun abonnement.
